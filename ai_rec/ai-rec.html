[컴퓨터구조론] CPU 정의

다음 중 중앙 처리 장치(CPU)의 가장 핵심적인 역할은 무엇인가?

  1. 데이터를 영구적으로 저장한다.
  2. 입출력 장치와 데이터를 주고받는 것을 관리한다.
  3. 프로그램의 명령어를 해독하고 실행한다.
  4. 컴퓨터 시스템의 전원을 공급한다.

CPU를 구성하는 주요 요소가 아닌 것은?

  1. 산술 논리 장치 (ALU)
  2. 제어 장치 (Control Unit)
  3. 레지스터 (Register)
  4. 주기억 장치 (Main Memory)

CPU는 컴퓨터 시스템에서 어떤 역할을 수행하며 '컴퓨터의 두뇌'라고 비유되는가?

  1. 데이터를 저장하고 관리하는 역할
  2. 사용자 인터페이스를 담당하는 역할
  3. 프로그램의 연산과 제어를 담당하는 역할
  4. 그래픽 및 영상 처리를 전문으로 하는 역할

정답: ③

해설: CPU는 프로그램에 담긴 명령어들을 순서대로 읽어와 그 의미를 파악(해독)하고, 해당 명령어에 따른 필요한 연산이나 동작을 실제로 수행(실행)하는 핵심 장치입니다. 다른 보기들은 CPU의 주된 역할이거나 CPU가 직접 수행하는 기능이 아닙니다.

정답: ④

해설: 주기억 장치(Main Memory)는 CPU 외부에 위치하여 프로그램과 데이터를 저장하는 장치입니다. 산술 논리 장치, 제어 장치, 레지스터는 CPU 내부에 포함된 중앙 처리 장치의 주요 구성 요소입니다.

정답: ③

해설: CPU는 컴퓨터 프로그램의 명령어에 따라 데이터를 계산하거나 이동시키고(연산), 프로그램의 실행 흐름을 관리하고 각 장치에 지시를 내리는(제어) 역할을 수행합니다. 이러한 핵심적인 역할 때문에 컴퓨터의 '두뇌'로 비유됩니다.
